Installing and Removing SIPcon1

Requirements

SIPcon1 requires the following software to be installed:
  • PC, at least 500 MHz CPU
  • Internet connection (if a firewall is in use, it has to be SIP capable)

Platforms

SIPcon1 has been successfully tested on the following platform:
  • Windows XP
As SIPcon1 is a Java client, it should work on any platform that supports Java. In order to get it work on other platforms, the Java Media Framework (JMF) Cross-Platform Version has to be downloaded and installed from http://java.sun.com/jmf.

Note! Sending audio is not possible with the cross-platform version of JMF!

Installing and Running

SIPcon1 installation program is written in Java, it can be run on any platform that supports Java. Java Runtime Environment as well as the libraries for Java Media Framework with Windows Performance Pack are included and automatically installed when using the Windows installation program.

Windows Platforms

Execute the installation program named SIPcon1-installer.exe.

At installation time, icons on the desktop, in the start menu, and in the startup folder may be automatically installed. After completing the installation, run the application by using one of the icons chosen at install time.

Linux Platforms

  1. Make sure Java Runtime Environment is properly installed. If JRE is not installed on your system, download the latest version from http://java.sun.com.
  2. Install Java Media Framework Linux Performance Pack, it can be downloaded from http://java.sun.com/jmf.
  3. Execute the installation package file named SIPcon1Installer.jar.
Note! SIPcon1 has been developed to function well on Windows platforms. How the application handles other platforms is not tested.

During the installation process, you will be able to specify the location where the application should be installed. Run the application by using the SIPcon1-crossPlatform.sh shell script file.

Other Platforms

  1. Make sure Java Runtime Environment is properly installed. If JRE is not installed on your system, download the latest version from http://java.sun.com.
  2. Install Java Media Framework Cross-Platform Version, it can be downloaded from http://java.sun.com/jmf.
  3. Execute the installation package file named SIPcon1Installer.jar.
Note! Java Media Framework Cross-Platform Version does not support sending audio and video. When using other platforms than Windows or Linux, you will only be able to receive audio and video. Text will still work both ways. Furthermore, SIPcon1 has been developed to function well on Windows platforms. How the application behaves on other platforms is not tested.

Run the application by using the syntax in SIPcon1-crossPlatform.sh.

Removing

Windows Platforms

In order to uninstall SIPcon1 and all surrounding components, use "Add/Remove Applications" tools in the control panel or use the uninstall icon in the start menu.

Other Platforms

In order to uninstall SIPcon1, remove the directory where SIPcon1 is installed and all subdirectories. In order to uninstall Java Media Framework and Java Runtime Environment, refer to the manuals of those programs.
